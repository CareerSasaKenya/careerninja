What's Been Set Up

Your job listings now have dynamic Open Graph (OG) meta tags that will display custom thumbnails when shared on social media platforms like Facebook, LinkedIn, Twitter, and WhatsApp.

How to Test

  1. Deploy to Production: OG tags need to be tested on a live URL (localhost won't work for social media crawlers)
  2. Get a Job URL: Navigate to any job detail page and copy the URL
  3. Use Testing Tools: Paste your job URL into these validators:
    • Facebook Sharing Debugger
    • Twitter Card Validator
    • LinkedIn Post Inspector

What Each Job Share Will Show

  • Title: Job Title at Company Name
  • Description: Job description snippet with location
  • Image: Custom thumbnail with job details on a gradient background

API Endpoints Created

  • /api/og/job/[id] - Generates dynamic SVG thumbnail for each job
  • /og-image.svg - Default site thumbnail
üí° Pro Tip: After deploying, if you update a job's details, you may need to use the Facebook Debugger's "Scrape Again" button to refresh the cached thumbnail.

Files Modified/Created

  • app/layout.tsx - Added default OG tags
  • app/jobs/[id]/layout.tsx - New layout for job pages
  • app/jobs/[id]/metadata.ts - Metadata generation logic
  • app/api/og/job/[id]/route.ts - API route for thumbnails
  • public/og-image.svg - Default site thumbnail
